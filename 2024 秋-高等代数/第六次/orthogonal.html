Problem 1 以下是一些关于 Rn 中向量内积的小题. 任选两题完成, 其中一题需要是 3. 或 4..

  • 依通常定义, 点乘 ⟨u,v⟩=uT⋅v 关于两个输入都是线性的.

  • 记 ∥v∥=⟨v,v⟩.

  1. 给定 ∥u∥≤1 与 ∥v∥≤1, 证明: 1−∥u∥2⋅1−∥v∥2≤1−⟨u,v⟩.

  2. 证明: ∥u+v∥⋅∥u−v∥≤∥u∥2+∥v∥2.

  3. 证明: ⟨u,v⟩=0, 当且仅当 ∥u∥≤∥u+c⋅v∥ 对一切 c∈R 成立.

  4. 证明: 任意给定 u≠0, 则对一切 ∥v∥=1 均有 ∥u−(∥u∥−1⋅u)∥≤∥u−v∥. 换言之, 球面上距 u 最近处恰是 u 的单位化向量.

  5. 表述并证明高中所学的极化恒等式.

  6. 表述并证明高中所学的平行四边形恒等式.

Problem 2 假定 n≥1. 称 P∈Rn×n 是 Rn 上的一个投影矩阵, 当且仅当 P2=P=PT.

  • 题 2. 与 3. 在补空间意义下等同, 完成其中一题即可; 4. 与 5. 同理; 6. 与 7. 亦同理.

  1. 若 P 是投影矩阵, 当且仅当存在正交矩阵 Q 使得 Q−1PQ=(IrOOO). (如果不清楚正交矩阵, 此题可以略过.)

  2. 证明: 投影矩阵和子空间双射对应, 具体的对应方式可以是列空间 P↔1:1C(P).

  3. 证明: 投影矩阵和子空间双射对应, 具体的对应方式可以是零空间 P↔1:1N(P).

  4. 任意给定 v≠0, 找到 P 使得 C(P)=span(v).

  5. 任意给定 v≠0, 找到 P 使得 N(P)=span(v).

  6. 给定 R5 中的列向量 S={(4,3,3,1,1),(6,2,2,2,1)}, 找到 P∈R5×5 使得 C(P)=span(S).

  7. 给定 R5 中的列向量 S={(4,3,3,1,1),(6,2,2,2,1)}, 找到 P∈R5×5 使得 N(P)=span(S).

备注: 计算得 ∥(4,3,3,1,1)∥=6, 以及 ∥(6,2,2,2,1)∥=7.

Problem 3 计算示例 (最小平方法).

  • 如果你想要一些简单的参考资料, 可以查看这篇笔记.

  • 中英日名词对照: 最小平方法, Least Squares Method, 最小二乗法.

  1. 给定 A=(ai,j)∈Rm×n 与 b=(bj)∈Rn. 记 x=(xi)∈Rm, 则

    F=∥Ax−b∥2=∑i=1m(∑j=1nai,jxj−bi)2.

    任给定指标 1≤j0≤n, 假设所有 xj (j≠j0) 均是常量, 仅 xj0 是变量. 通过下式计算二次函数 F=F(xj0) 导数为零的点

    dFdxj0=ddxj0[∑i=1m(∑j=1nai,jxj−bi)2]=0.

    • F 何时是二次函数? 我们需要排除一些平凡情形, 请稍作说明.

  2. 假设上一问中 F′(xj0)=0 的解是 xj0=Xj0. 记解向量 X=(Xi)∈Rm. 证明 ATAX=ATb.

  3. (自主思考, 这不是一个问题) 上式合并了有唯一解, 有无穷解, 以及无解这三种情况. 请区分, 讨论这些情况.

  4. 使用最小平方法找到一条抛物线 y=a+bx+cx2, 使得该抛物线可以尽可能地拟合以下所有点:

    {(−2,4),(−1,2),(0,1),(2,1),(3,1)}.

    提示: 可以考虑方程

    (1x1x121x2x22⋮⋮⋮1xnxn2)⋅(abc)=(y1y2⋮yn).

    请用严谨的数学语言解释这一所谓的拟合 (应当先定义点到抛物线的距离.).
